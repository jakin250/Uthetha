# Uthetha Word Counter - Quick Start Guide

## Installation & Setup

### Step 1: Open the Website
- Navigate to the `Front End` folder
- Double-click `index.html` to open it in your default web browser
- Or right-click and select "Open with" to choose a specific browser

### Step 2: Allow Full Features
- Ensure JavaScript is enabled (default in all modern browsers)
- Some features like "Copy to Clipboard" may require HTTPS or localhost

## Basic Workflow (30 seconds)

1. **Paste Text** → Click in the text area and paste your content
2. **Check Stats** → See word/character/sentence/paragraph counts update automatically
3. **Search Keywords** → Type keywords in the search box (comma-separated)
4. **Highlight Results** → Click "Search" to highlight keywords with colors
5. **Analyze Density** → Go to Keyword Density section and click "Calculate Density"

## Common Tasks

### Task: Count words in my article
1. Paste your article in the text input
2. Look at the "Words" stat in the Statistics section
3. Done! ✓

### Task: Find all mentions of specific terms
1. Enter keywords separated by commas: `keyword1, keyword2, keyword3`
2. Click "Search"
3. All occurrences are highlighted in different colors
4. See the count for each keyword in the search info box

### Task: Find overused words
1. Paste your text
2. Go to "Keyword Density Tracker"
3. Set minimum occurrences to 2-3
4. Click "Calculate Density"
5. See which words you use most frequently

### Task: Check my sentence structure
1. Paste text
2. Look at "Sentences" count
3. Look at "Avg Words/Sentence" metric
4. Good readability = 15-20 words per sentence average

## Features at a Glance

| Feature | Location | What it does |
|---------|----------|--------------|
| Word Count | Statistics | Counts words in your text |
| Character Count | Statistics | Counts all characters |
| Sentence Count | Statistics | Detects sentences by . ! ? |
| Paragraph Count | Statistics | Detects paragraphs by blank lines |
| Search Keywords | Search & Highlight | Finds keywords anywhere in text (not just side-by-side) |
| Color Highlighting | Search & Highlight | Colors each keyword differently |
| Keyword Density | Keyword Density | Shows which words you use most |

## Pro Tips

✓ **Search Tip**: Use exact keywords without extra punctuation for best results
✓ **Density Tip**: Lower the minimum occurrences to see more keywords
✓ **Writing Tip**: Aim for 15-20 words per sentence for readability
✓ **Analysis Tip**: Copy-paste long articles section by section for detailed analysis

## Troubleshooting

**Q: Page won't load**  
A: Make sure all files (index.html, style.css, script.js) are in the same folder

**Q: Styles look wrong**  
A: Clear your browser cache (Ctrl+Shift+Delete) and refresh

**Q: Copy button not working**  
A: Some browsers require HTTPS for clipboard access. Try using Ctrl+A then Ctrl+C

**Q: Numbers not updating**  
A: Check if JavaScript is enabled. Refresh the page.
