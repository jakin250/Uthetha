What gTTS can and cannot do

  • True male and female voice switching is not exposed by this library.
  • Slow mode is included here as a practical alternative for clearer delivery.

How It Works

Simple web flow

  • Choose one of the currently supported language or dialect options.
  • Paste up to {{ max_text_length }} characters into the text box.
  • Generate audio, preview it in the browser, then download the MP3.

Important Note

Internet access is required

gTTS sends requests to Google Text-to-Speech to create the audio file, so the app needs an active internet connection when users generate speech.
